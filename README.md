# NihilXD's Unraid Templates

Community Applications templates for Unraid.

## Containers

### [Windrose Dedicated Server](windrose-server.xml)
Host a persistent Windrose world 24/7 so your crew can sail, build, and plunder without needing a host online. Features auto-updating via SteamCMD, full configuration via the Unraid UI, and working multiplayer with no port forwarding required.
